# My-Portfolio-Website-Advanced-Version-
Responsive portfolio website with glassmorphism design, dark mode toggle, and JavaScript form validation and Mobile-first responsive portfolio showcasing projects, skills, animations, and interactive features using vanilla JavaScript.


This project is a responsive personal portfolio website designed to showcase:
  - About Me section
  - Skills with animated progress bars
  - Projects section
  - Contact form with validation
  - Dark / Light mode toggle
  - Mobile-first responsive design

The website follows modern UI principles including glassmorphism design and smooth animations.

Features: Responsive layout (Mobile-first)
          CSS Grid & Flexbox layout
          Glassmorphism UI design
          Dark/Light theme toggle
          Typing animation effect
          Scroll-based skill animation
          Form validation using JavaScript
          Hamburger menu for mobile
          Smooth transitions and hover effects
